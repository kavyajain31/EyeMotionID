# EyeMotionID — Deep Learning Eye Movement Biometrics

A behavioral biometric authentication system that identifies users
using dynamic eye movement patterns.

## Tech Stack
Python | PyTorch | OpenCV | MediaPipe | NumPy | Pandas | Matplotlib | Scikit-learn

## Architecture
Input Eye Video → Eye Detection → Blink/Gaze Tracking
→ CNN (ResNet18) → LSTM Temporal Modeling → User Identification

## Folder Structure
- data/        → raw and processed datasets
- models/      → saved model weights
- notebooks/   → Jupyter experiment notebooks
- src/         → source code modules
- results/     → evaluation plots and reports
- docs/        → literature review and notes
